# Git_Hired: Career Forge AI (Champion Theme)

Git_Hired is a premium AI-powered career preparation platform designed to help developers master their job search. Featuring a luxurious "Champion" aesthetic, the application provides advanced tools for resume analysis and mock interviews.

The application features a high-end, dark-mode interface inspired by Awwwards-winning designs:

- **Palette**: Deep Black (`#050505`), Champion Gold (`#D4AF37`), Silver, and White.
- **Typography**: `Oswald` (Headings) & `Manrope` (Body) for a modern, bold look.
- **Animations**: Silky smooth transitions and micro-interactions powered by `framer-motion`.

## ‚ú® Key Features

### 1. AI Resume Scanner

- **Drag & Drop Interface**: Upload PDF resumes easily.
- **ATS Analysis**: Get an instant compatibility score (0-100).
- **Detailed Feedback**: Receive actionable insights to improve your CV.
- **Visualizations**: Animated score bars and success metrics.

### 2. AI Mock Interview Simulation

- **Role Selection**: Choose from Frontend, Backend, Full Stack, DevOps, or PM roles.
- **Difficulty Levels**: Normal vs. Expert protocols.
- **Interactive Chat**: Real-time conversation with an AI interviewer.
- **Performance Report**: Comprehensive post-interview analysis with grading (S/A/B/C).

### 3. User Profile & Gamification

- **XP System**: Earn experience points for every resume scan and interview.
- **Leveling**: Track your progress from "Novice" to "Champion".
- **History Logs**: Review past performance and track improvement over time.
- **Responsive Stats**: Visual breakdown of your activity.

## üöÄ Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/your-repo/career-forge-ai.git
   ```

2. Install dependencies

   ```bash
   cd client-retro
   npm install
   ```

3. Start the development server

   ```bash
   npm run dev
   ```

4. Build for production
   ```bash
   npm run build
   ```
