## üíæ SVG Export Best Practices

When exporting SVGs:

```
‚úÖ DO:
- Export as SVG (not PNG converted to SVG)
- Use scalable units (not fixed pixels for main elements)
- Keep stroke width consistent
- Use path data instead of strokes when possible
- Remove unnecessary metadata
- Test at multiple sizes

‚ùå DON'T:
- Embed raster images inside SVG
- Use overly complex paths
- Include unnecessary color spaces
- Forget to optimize file size
- Use absolute pixel sizes for responsive elements
```
